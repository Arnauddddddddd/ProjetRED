Salut, quelques informations pour lancer le jeu :

1- Ouvrir le dossier ProjetRED avec VScode
2- Executez la commande "/cd src" puis "/go run ." pour lancer le jeu !


informations du jeu :

    But: Atteindre le boss final dans la dernière zone du jeu. Vous aurez le choix d'une classe au lancement du jeu, chaque classe possède ses propre avantages.

    Pour aider à évoluer, il va falloir farmer des monstres pour récupérer des items dans la boutique : les plus importants étant : 
        -l'étoile qui augmente vos dégats et votre vie.
        -la clef qui permet d'ouvrir la porte vers le désert

    Les combats se déclenchent lorsque vous êtes proche d'un ennemi, vous aurez le choix :
        1- d'attaquer normalement 
        2- utiliser en combat un objet de l'inventaire
        3- utiliser votre capacité spécial de classe utilisable qu'une fois par combat

    Touches:
        -déplacements : ZQSD
        -ouvrir/fermer l'inventaire : E
        -ouvrir/fermer le Shop : E (il faut être proche du shop)
        -déconseillé pour tester le jeu : Touche de triche : Rester appuyer sur Shift permet d'accroître la vitesse et de traverser les murs.

    Infos:
        lors des combats, vous pouvez spammer le bouton clique gauche pour passer les textes.
